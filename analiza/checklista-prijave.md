# Checklista prijave

## 1. Odabir tocke

- [x] odabrana je tocna tocka javnog poziva
- [x] provjereno je da se za svaku tocku radi zasebna prijava
- [x] pripremljena je zasebna omotnica za svaku tocku

Komentar:
- prijave su pripremljene zasebno za tocke `15-308`, `15-508` i `15-510`, a slanje je evidentirano 31. ozujka 2026.

## 2. Podaci za obrazac

- [x] upisano ime i prezime / naziv
- [x] upisana adresa / sjediste
- [x] upisan telefon
- [x] upisana e-mail adresa
- [x] upisan OIB
- [x] upisana tocka javnog poziva
- [x] upisan ponudeni godisnji iznos zakupnine bez PDV-a
- [x] upisan rok zakupa u godinama
- [x] upisano mjesto i datum
- [x] obrazac potpisan

Komentar:
- obrasci `02` su popunjeni za sve tri tocke i poslani preporucenom postom.

## 3. Obvezna dokumentacija za pasarenje

- [x] potvrda o prebivalistu ili izvadak iz odgovarajuceg registra
- [x] potvrda Porezne uprave o nepostojanju duga
- [x] gospodarski program koristenja sumskog zemljista
- [x] izjava o prihvacanju uvjeta javnog poziva
- [x] potvrda o upisu u Upisnik poljoprivrednika
- [x] izlistaj stoke
- [x] potvrda o broju uvjetnih grla
- [x] potvrda o evidentiranim povrsinama u sustavu ARKOD
- [x] ugovor o zakupu, ako se koristi kriterij iz clanka 9. stavka 1. tocke 1.
- [x] postojeci ugovori s Hrvatskim sumama su prilozni ako se na njima temelji pravo prednosti ili status dosadasnjeg zakupnika
- [x] evidentiran je odgovor Hrvatskih suma / racunovodstva o stanju obveza po dosadasnjim ugovorima
- [x] ocitovanje Hrvatskih suma o stanju obveza priloziti u svaku omotnicu kao dodatni dokaz urednog izvrsavanja obveza
- [x] dokaz o vlasnistvu ili posjedu zemljista, ako se trazi pravo prednosti iz clanka 9. stavka 2.
- [x] izjava o obvezi izjednacavanja potrebnog broja uvjetnih grla, ako je trazena povrsina do 10% veca od trenutnih potreba

Komentar:
- za tocke `15-308`, `15-508` i `15-510` potvrdeno je da izjava o izjednacavanju uvjetnih grla nije potrebna, pa je uvjet zatvoren kao provjeren.
- postojeci ugovori i ocitovanje o stanju obveza evidentirani su kao osnova za argumentaciju kontinuiteta i urednog izvrsavanja obveza.

## 4. Provjere prije slanja

- [x] sva dokumentacija nije starija od 30 dana od dana objave javnog poziva
- [x] izjava o prihvacanju uvjeta javnog poziva je ispunjena za pravnu osobu i potpisana od ovlastene osobe
- [x] pregledana je Obavijest o obradi osobnih podataka i potvrdeno je da dostava osobnih podataka odgovara prijavi
- [x] provjeren je odnos povrsine i broja uvjetnih grla
- [x] provjereno postoji li osnova za pravo prednosti
- [x] provjereno postoji li dokaz dosadasnjeg zakupa i urednog ispunjavanja obveza
- [x] provjereno je stanje obveza prema Hrvatskim sumama i evidentirano da postoji pretplata od 0,66 EUR prema zaprimljenom odgovoru racunovodstva
- [x] u prijavi i popratnoj argumentaciji je jasno navedeno postojanje dosadasnjih ugovora s Hrvatskim sumama, ako se na tome temelji prednost
- [x] provjerena je lokalna povezanost ako se na nju racuna
- [x] provjeren je ponudeni iznos zakupnine

Komentar:
- zavrsna dokumentacijska kontrola je dovrsena prije slanja; operativni status je od 31. ozujka 2026. prebacen sa pripreme na pracenje postupka.
